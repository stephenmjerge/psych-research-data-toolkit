
# PRDT Roadmap — Psych Research Data Toolkit
## Purpose
Clean, anonymize, and standardize mental‑health datasets reproducibly.

## Phase 1
- Implement cleaning pipeline.
- Publish first OSF dataset (synthetic Dataset A demo bundle attached to lab hub DOI: https://doi.org/10.17605/OSF.IO/BX76K with PRDT component https://osf.io/qs8ag/ and Dataset A sub-component https://osf.io/n4buw/; replicate with live Dataset A once approved).
- Write validation + descriptives notebook.
- Tag v0.5 release.
- Document Dataset A cleaning profile (configs/dataset_a.toml + README note) — added Nov 13, 2025.
- Capture Dataset A descriptives/plots for OSF handoff (docs/assets/dataset-a-demo + docs/dataset_a.md) — added Nov 13, 2025.
