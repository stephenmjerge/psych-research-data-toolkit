# PRDT Dataset A — Demo Bundle (OSF Deposit)

This deposit contains the reproducible outputs generated by the Psych Research Data
Toolkit (PRDT) for a synthetic Dataset A. The dataset models PHQ-9, GAD-7, and
PCL-5 survey fields and is used to demonstrate PRDT’s cleaning and anonymization
workflow. All data are fully synthetic and contain no PHI.

- **Generated:** 2025-11-14  
- **Pipeline Script:** `scripts/run_dataset_a_osf_demo.sh`  
- **Input:** Synthetic demo file (`dataset_a_demo.csv`)  
- **Manifest:** `run_manifest_20251115T021847Z.json`  
- **PRDT Version:** 0.1.0  

## Contents

- **interim_clean.csv** – fully cleaned, anonymized Dataset A output  
- **report.json** – cleaning summary and validation metrics  
- **alerts.json** – detected anomalies, missingness flags, and range checks  
- **data_dictionary.csv** – variable definitions and scoring metadata  
- **plots/** – distribution histograms, missingness visualizations, trend plots  
- **dataset_a.toml** – configuration used for this run  
- **run_manifest_20251115T021847Z.json** – full reproducibility manifest  

## Purpose

This deposit demonstrates transparent, reproducible data-cleaning practices for
mental-health survey datasets, consistent with open-science standards. It serves as a
public, citable example of PRDT’s capabilities.
