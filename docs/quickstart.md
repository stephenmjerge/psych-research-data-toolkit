# PRDT Quickstart (Non-Technical)

### 1. Download and Install
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements-lock.txt
pip install dist/prdt-0.1.4-py3-none-any.whl  # or pip install -e . if you cloned the repo
python -m build  # optional: regenerate wheels for other machines
```

### 2. Set your anonymization key
```bash
export PRDT_ANON_KEY="$(openssl rand -hex 64)"
# or add PRDT_ANON_KEY=... to a local .env file (PRDT reads it automatically)
```

### 3. Run the sample config
```bash
prdt --config configs/anxiety.toml
```

### 4. Review outputs
- `outputs/anxiety-profile/interim_clean.csv`
- `report.json`, `alerts.json`, `phi_quarantine.csv`, `drift.json`
- Visuals: `hist_*`, `missingness.png`, `scale_summary.png`, `scale_items_<scale>.png`

### 5. Use your own CSV
1. Copy the profile: `cp configs/anxiety.toml configs/my_clinic.toml`
2. Edit:
   - `input` and `outdir`
   - `[prdt.score]` definitions for your scales
   - `[prdt.schema]` required columns and ranges
   - `[prdt.phi]` keywords to match your charting vocabulary
3. Run `prdt --config configs/my_clinic.toml`

### 6. Share safely
- Send only the files under `outputs/...` (never the raw CSV or `.env`).
- Mention that all outputs were generated by PRDT version `X` using config `Y`; link to the manifest for proof.

### Headless / CI runs
- Skip plots: `export PRDT_DISABLE_PLOTS=1`
- Redirect Matplotlib caches to writable folders when `$HOME` is locked down:
  ```bash
  export MPLCONFIGDIR=$PWD/.cache/mpl
  export XDG_CACHE_HOME=$PWD/.cache/xdg
  ```

That’s it—copy/paste these commands and you’ll have reproducible, anonymized reports in minutes.
